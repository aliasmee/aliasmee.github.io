我的新玩伴-树莓派2

@aliasmee · Jun 24, 2018 · 1 min read

接着上篇的，本篇介绍了桌面版本的安装，并设置vnc ，用于远程桌面连接。另外给大家看自己动手给树莓派盖的屋子。哈哈

安装VNC

确认vncserver 已经安装：

sudo apt-get install realvnc-vnc-server realvnc-vnc-viewer

启动并创建自启动文件

systemctl start vncserver-x11-serviced

systemctl enable vncserver-x11-serviced

设置vnc认证

修改配置文件/root/.vnc/config.d/vncserver-x11，将systemauth 修改为vncauth Authentication=VncAuth

运行以下指令，根据提示设置一个vnc登录认证密码:

vncpasswd -service
